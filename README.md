## Getting Started


### Prerequisites

Python 3.7.3


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/tslr/ml
   ```
2. Install PIP packages
   ```sh
   pip install
   ```



<!-- USAGE EXAMPLES -->
## Usage

-> Run the flask app
   ```sh
   cd app
   python app.py
   ```

-> Run as a docker container
   ```sh
   ./start.sh
   ```
The api runs at localhost:5000/.
